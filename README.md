# Implementación de estructuras de datos en Java

Trabajo práctico de la materia Algoritmos y Estructuras de Datos (UBA).

## Descripción
Implementación de distintas estructuras de datos y clases auxiliares en Java, incluyendo:

- Heap con handles
- Pila
- Modelado de estudiantes y cálculo de notas finales
- Resolución de consignas utilizando estructuras propias

El objetivo del proyecto es aplicar conceptos de estructuras de datos, complejidad algorítmica y programación orientada a objetos.

## Estructura del proyecto
- Heap.java: implementación de heap con handles
- Handle.java: manejo de referencias dentro del heap
- Pila.java: implementación de pila
- Estudiante.java y NotaFinal.java: modelado de datos
- Edr.java: resolución de consignas del trabajo práctico

Lenguaje: Java
